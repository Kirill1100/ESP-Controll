# ESP-Controll
ESP32-C3 Project By KyrexX 
To do this project you need
1) ESP32-C3
2) OLED Display 128x64 with 4 buttons
3) Battery 3.3volts (optional)
4) switch (optional)

Pins to connect display to esp32c3:
OLED_GND gnd 
OLED_VCC vcc 
OLED_SCL 4 
OLED_SDA 3 
BTN_UP   6 
BTN_DOWN 5 
BTN_OK   7 
BTN_BACK 8
